# Mi Primera App en Angular

## Descripción del proyecto

Este proyecto fue realizado como práctica inicial de Angular. Se trabajó en la creación de una aplicación básica utilizando Angular CLI, modificando componentes y aplicando interpolación de variables en la vista.

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML
- CSS
- Node.js
- Angular CLI

## Instalación del proyecto

Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Ingresar a la carpeta:

```bash
cd mi-primera-app-angular
```

Instalar dependencias:

```bash
npm install
```

Ejecutar la aplicación:

```bash
ng serve
```

Abrir en el navegador:

```txt
http://localhost:4200
```

## Funcionalidades realizadas

- Creación de proyecto Angular
- Cambio de título
- Agregado de información personal
- Uso de interpolación
- Visualización dinámica de variables {{nombre}}

## Estructura principal

### src/app

Carpeta principal de componentes y lógica de la aplicación.

### app.component.ts

Contiene la lógica del componente principal.

### app.module.ts

Módulo principal donde se configura la aplicación.

### assets/

Carpeta para imágenes y recursos estáticos.

### environments/

Configuraciones de entorno.
